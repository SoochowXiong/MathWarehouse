# MathWarehouse 数学资源库
欢迎来到数学资源库！这是[Xiong](https://github.com/SoochowXiong)于2024年10月26日创建的一个仓库，旨在为苏州大学数学科学学院的学生和数学爱好者提供重要的数学资源和参考资料。希望通过此平台，帮助提升数学知识理解和应用的水平，并鼓励协作学习。
